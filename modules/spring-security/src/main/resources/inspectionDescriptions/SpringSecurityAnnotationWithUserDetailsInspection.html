Unresolved Spring beans referenced in Spring Security annotations.

Examples:

  • 
        @WithUserDetails(value = "user@example.com", userDetailsServiceBeanName = "wrongService")
        // "Cannot resolve bean 'wrongService'" - 'wrongService' is not defined in the Spring model
    
  • 
        @WithUserDetails(value = "admin", userDetailsServiceBeanName = "notDetailsService")
        // "Bean must be of 'org.springframework.security.core.userdetails.UserDetailsService' type" - "notDetailsService" bean is not of 'UserDetailsService' type
