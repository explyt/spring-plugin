Reports duplicated RequestMapping endpoint

Example:

Reports "Duplicated endpoint 'first'" and "Duplicated endpoint 'second'" 

    @Controller("some")
    class SomeController {

        @GetMapping("/some/uri") //Duplicated endpoint 'second'
        public String first() {
            return "first";
        }

        @GetMapping("/some/uri") //Duplicated endpoint 'first'
        public String second() {
            return "second";
        }

    }
