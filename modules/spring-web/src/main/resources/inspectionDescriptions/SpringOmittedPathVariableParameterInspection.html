Reports "@PathVariable parameters and the URL path are not fit each other" when path parameters are declared in a URL path of @RequestMapping annotation but omitted in @PathVariable parameters and vise versa.

Example:

"@PathVariable parameters and the URL path are not fit each other" will be reported on 'id' parameter if there is no @PathVariable argument with the same name Quick Fix adds omitted argument 

@GetMapping("/api/employees/{id}")
@ResponseBody
public String getEmployeesById(/*@PathVariable String id*/) {
	return "ID: " + id;
}

Example:

"@PathVariable parameters and the URL path are not fit each other" will be reported on 'renamedId' parameter if there is no URL path argument with the same name 

//@GetMapping("/api/employees/{renamedId}")
@ResponseBody
public String getEmployeesById(@PathVariable("renamedId") String id) {
	return "ID: " + id;
}
