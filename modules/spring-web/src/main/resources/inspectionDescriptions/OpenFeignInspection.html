Feign can handle load balancing via service registration and service discovery itself.

In the @FeignClient annotation the String value is an arbitrary client name, which is used to create a Spring Cloud LoadBalancer client.

See netflix-feign-starter 


@FeignClient("stores")
public interface StoreClient {
    @LoadBalanced // @LoadBalanced annotation does not need to be used in the FeignClient interface
    @RequestMapping(method = RequestMethod.GET, value = "/stores")
    List<Store> getStores();
}

If you want to load balance services registered in a service registry (for example, Eureka), you only need to specify the service name in the @FeignClient via the name (or value) parameter.
