Reports incorrect @Scheduled expressions:

  • Spring Scheduled must not be empty
  • Scheduled contain correct schedule interval

Examples:


  // correctly configured
  @Scheduled(cron="* * * * * *")
  public void scheduledMethod() {...}

  //correctly configured
  @Scheduled(fixedRate=100)
  public void scheduledMethod() {...}

  //correctly configured
  @Scheduled(fixedDelayString="${property.value}")
  public void scheduledMethod() {...}

  //correctly configured
  @Scheduled(fixedDelayString="PT02S")
  public void scheduledMethod() {...}

  //incorrectly configured: empty
  @Scheduled()
  public void scheduledMethod() {...}

  //incorrectly configured: invalid cron
  @Scheduled(cron="* *")
  public void scheduledMethod() {...}

  //incorrectly configured: invalid string
  @Scheduled(fixedDelayString="1abc")
  public void scheduledMethod() {...}
