Reports wrong placement for @ConditionalOnProperty, @ConditionalOnClass, @ConditionalOnMissingClass, @ConditionalOnBean, @ConditionalOnMissingBean annotations.

Example:

"Should annotate bean/component" will be reported on '@ConditionalOnMissingBean' annotation if class is not annotated with '@Component' 

  //@Component
  @ConditionalOnMissingBean
  public class MyComponent {
  }

Example:

"Should annotate bean/component" will be reported on '@ConditionalOnProperty' annotation if method is not annotated with '@Bean' 

  @Configuration
  public class MyConfiguration {
    //@Bean
    @ConditionalOnProperty("some.property")
    SomeClass notABean() { ... }
  }
