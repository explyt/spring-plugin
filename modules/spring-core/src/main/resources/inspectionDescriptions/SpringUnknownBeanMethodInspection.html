Reports 'Cannot find method' on initMethod and destroyMethod parameters of the @Bean annotation.

Example:

If BeanA class doesn't contain public 'unknownMethod' method, "Cannot find method" will be reported on 'unknownMethod' 

  public class BeanA {...}

  @Configuration
  public class BeanMethodConfiguration {
     @Bean(initMethod="unknownMethod" )
     public BeanA beanA() {...}
  }
