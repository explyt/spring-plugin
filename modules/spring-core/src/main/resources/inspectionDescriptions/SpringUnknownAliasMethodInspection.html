Reports unresolved method references on value and attribute parameters of the @AliasFor annotation.

Example:

"Cannot resolve attribute" will be reported on 'wrong' expression if SomeAnnotation doesn't contain 'wrong' attribute 

  public @interface SomeAnnotation {...}

  @Component
  @SomeAnnotation
  public @interface Another {
     @AliasFor(annotation = SomeAnnotation.class, attribute = "wrong")
     String attributeName() default "";
  }
