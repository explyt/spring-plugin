Reports wrong placement for @Profile annotation.

Example:

"Profile should annotate bean/component" will be reported on '@Profile' annotation if class is not annotated with '@Component' 

  //@Component
  @Profile("myProfile")
  public class MyComponent {
  }

Example:

"Profile should annotate bean/component" will be reported on '@Profile' annotation if method is not annotated with '@Bean' 

  @Configuration
  public class MyConfiguration {
    //@Bean
    @Profile("myProfile")
    SomeClass notABean() { ... }
  }
