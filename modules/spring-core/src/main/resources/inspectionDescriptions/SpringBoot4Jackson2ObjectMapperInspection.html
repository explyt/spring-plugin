Reports an injection point (autowired field, constructor parameter of a Spring bean, or @Bean method parameter) whose type is the Jackson 2 com.fasterxml.jackson.databind.ObjectMapper in a Spring Boot 4+ project.

Spring Boot 4 ships Jackson 3: JacksonAutoConfiguration registers only Jackson 3 beans (the primary mapper is tools.jackson.databind.json.JsonMapper), and no Jackson 2 ObjectMapper bean exists. Such an injection still compiles because Jackson 2 usually remains a transitive dependency, but the context fails at runtime with NoSuchBeanDefinitionException.

The inspection is skipped when the project opted into the deprecated Jackson 2 support (spring-boot-jackson2), which auto-configures Jackson 2 beans again.

Example:


@RestController
public class WebhookController {
    ObjectMapper objectMapper; // reported: no Jackson 2 bean; inject tools.jackson.databind.json.JsonMapper
}

The quick-fix replaces the injected type with tools.jackson.databind.json.JsonMapper.
