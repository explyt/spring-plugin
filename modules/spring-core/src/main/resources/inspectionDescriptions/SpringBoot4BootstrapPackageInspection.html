Reports the legacy BootstrapRegistry / BootstrapContext import locations in Spring Boot 4+ projects and offers a quick-fix that rewrites the import to the new org.springframework.boot.bootstrap package.

In Spring Boot 4.0 BootstrapRegistry, ConfigurableBootstrapContext and DefaultBootstrapContext (and related bootstrap types) moved from org.springframework.boot to org.springframework.boot.bootstrap. The inspection only triggers when Spring Boot 4+ is detected and the new class is resolvable on the classpath.

See the Spring Boot 4.0 Migration Guide.
