Reports declarations (fields, method parameters and return types) whose type is a BootstrapRegistry / BootstrapContext family type from the legacy org.springframework.boot package in Spring Boot 4+ projects, and offers a quick-fix that updates the import to the new org.springframework.boot.bootstrap package.

In Spring Boot 4.0 BootstrapRegistry, ConfigurableBootstrapContext and DefaultBootstrapContext (and related bootstrap types) moved from org.springframework.boot to org.springframework.boot.bootstrap. The visible type usage is highlighted (not the import statement, which the IDE usually folds away).

See the Spring Boot 4.0 Migration Guide.
