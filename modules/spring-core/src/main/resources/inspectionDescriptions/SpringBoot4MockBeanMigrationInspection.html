Reports Spring Boot's @MockBean / @SpyBean test annotations, which were removed in Spring Boot 4.0, and offers a quick-fix that replaces them with Spring Framework's @MockitoBean / @MockitoSpyBean (updating the import).

Only runs in a Spring Boot 4+ project.

Example:


// before
import org.springframework.boot.test.mock.mockito.MockBean;

@MockBean
private UserService userService;

// after
import org.springframework.test.context.bean.override.mockito.MockitoBean;

@MockitoBean
private UserService userService;

See the Spring Boot 4.0 Migration Guide.
