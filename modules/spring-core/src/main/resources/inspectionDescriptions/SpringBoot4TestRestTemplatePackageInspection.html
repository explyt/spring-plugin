Reports the legacy TestRestTemplate import location in Spring Boot 4+ projects and offers a quick-fix that rewrites the import to the new package.

In Spring Boot 4.0 TestRestTemplate moved from org.springframework.boot.test.web.client to org.springframework.boot.resttestclient (provided by the new spring-boot-resttestclient module). The inspection only triggers when Spring Boot 4+ is detected and the new class is resolvable on the classpath.

Example:


// before
import org.springframework.boot.test.web.client.TestRestTemplate;

// after
import org.springframework.boot.resttestclient.TestRestTemplate;

See the Spring Boot 4.0 Migration Guide.
