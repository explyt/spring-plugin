Reports interchangeable attributes used simultaneously.

Example:

When interchangeable attributes are defined in annotation like this: 

public @interface EventListener {
	@AliasFor("classes")
	Class[] value() default {};

	@AliasFor("value")
	Class[] classes() default {};
}
Simultaneously usage of these attributes will cause error: 

@EventListener(classes = {Foo.class}, value = {Bar.class})
public void foo() {}
