Reports Spring Boot configuration properties that were renamed or removed in Spring Boot 3.0, and offers a quick-fix that renames the key (.properties) or removes a no-longer-supported key.

The inspection only runs on Spring configuration property files of a Spring Boot 3+ project.

Examples:


# renamed
server.max-http-header-size=8KB        # -> server.max-http-request-header-size
spring.redis.host=localhost            # -> spring.data.redis.host
spring.data.cassandra.port=9042        # -> spring.cassandra.port

# removed (no longer supported)
spring.jpa.hibernate.use-new-id-generator-mappings=true
spring.session.store-type=redis
