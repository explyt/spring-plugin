The @EventListener inspection checks two conditions:

  1. Method accessibility: It ensures that methods annotated with @EventListener are public for proper event handling.
  2. Parameter count: It verifies that annotated methods have a maximum of one parameter, conforming to the event handling requirements.

Example:


import org.springframework.context.event.EventListener;
import org.springframework.stereotype.Component;

@Component
public class ExampleListener {

    @EventListener // Method annotated with @EventListener must be public
    private void handleEvent(Object event) {
    }

    @EventListener // Method must have maximum one parameter
    public void processEvent(Object event, String additionalData) {
    }
}
