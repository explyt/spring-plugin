Reports a @SpringBootTest class that injects a web test client (MockMvc / TestRestTemplate) without the auto-configuration annotation that Spring Boot 4 now requires, and offers a quick-fix that adds it.

In Spring Boot 4.0 @SpringBootTest no longer auto-configures MockMvc / TestRestTemplate; the corresponding @AutoConfigureMockMvc / @AutoConfigureTestRestTemplate annotation must be added explicitly.

Example:


// before
@SpringBootTest
class MyTest {
    @Autowired MockMvc mockMvc;
}

// after
@SpringBootTest
@AutoConfigureMockMvc
class MyTest {
    @Autowired MockMvc mockMvc;
}

See the Spring Boot 4.0 Migration Guide.
